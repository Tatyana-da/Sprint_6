# Sprint_6

# Автотесты для сервиса «Яндекс.Самокат»

## Описание
Проект содержит автотесты для учебного сервиса «Яндекс.Самокат».  
Тесты написаны на Python с использованием Selenium, Page Object Model и Allure.

## Технологии
- Python 3.8+
- Selenium WebDriver
- pytest
- Allure
- Page Object Model

## Тестовые сценарии

### 1. Вопросы о важном (FAQ)
Проверка работы выпадающего списка в разделе «Вопросы о важном».  
Для каждого из 8 вопросов написан отдельный тест, который проверяет:
- При клике на вопрос открывается соответствующий текст ответа

### 2. Заказ самоката
Проверка позитивного сценария оформления заказа:
- Заполнение формы заказа (данные клиента + данные аренды)
- Проверка появления сообщения об успешном создании заказа
- Тесты запускаются с тремя наборами данных через разные кнопки заказа

### 3. Логотипы
- Проверка перехода на главную страницу при клике на логотип «Самоката»
- Проверка открытия страницы Дзена в новом окне при клике на логотип Яндекса
